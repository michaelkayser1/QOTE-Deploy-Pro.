┌───────────────────────────────────────────────────────────────────────────────┐
│ KEY METRICS                                                                   │
└───────────────────────────────────────────────────────────────────────────────┘

  Δθᵢ = |θᵢ - θᵢ₋₁|           Phase shift per step

  Δθ_total = Σᵢ Δθᵢ            Cumulative coherence metric

  COHERENCE ZONES:
    ✓ Safe:    Δθ_total < π      (green light - proceed)
    ⚠ Caution: π ≤ Δθ_total < 2π (yellow - monitor closely)
    ✗ Danger:  Δθ_total ≥ 2π     (red - pause & consolidate)

┌───────────────────────────────────────────────────────────────────────────────┐
│ QUICK API EXAMPLE (PYTHON)                                                    │
└───────────────────────────────────────────────────────────────────────────────┘

  from resona import ResonaMiddleware

  resona = ResonaMiddleware(
      delta_theta_safe=3.14,
      delta_theta_max=6.28,
      regulation_policy="adaptive"
  )

  for step in system.iterate():
      qote_state = resona.extract_phase(step.state)
      status = resona.monitor_coherence(qote_state)
      control = resona.regulate(status)

      if control == "Pause":
          step.halt()
          await human_review()
      elif control == "Consolidate":
          step.inject_prompt("Explain this step more clearly.")
      else:
          step.continue_()

┌───────────────────────────────────────────────────────────────────────────────┐
│ HUMAN–AI SYNCHRONIZATION (BIOMETRIC MODE)                                     │
└───────────────────────────────────────────────────────────────────────────────┘

  1. Measure human HRV (Polar H10 or similar)
  2. Extract dominant frequency f_HRV (typically ~0.1 Hz)
  3. Set AI rhythm: τ = 1 / f_HRV
  4. Compute phase-locking value (PLV) between AI Δθ and HRV
  5. Adjust τ dynamically to maximize PLV

  Target: PLV > 0.3 (significant synchronization)

┌───────────────────────────────────────────────────────────────────────────────┐
│ ADAPTIVE THRESHOLD TUNING                                                     │
└───────────────────────────────────────────────────────────────────────────────┘

  Δθ_max(t+1) = Δθ_max(t) + α · feedback(t)

  Where feedback ∈ {-1, 0, +1} from user:
    +1: "I could follow that, show me more"
     0: "Acceptable"
    -1: "Lost me, slow down"

  Typical α: 0.1 to 0.5
